# Data_Visualization_Correlation_Analysis
An exploratory data analysis project that applies box plot visualization and statistical correlation methods to identify patterns and relationships in real-world video game and health datasets.

## Overview

This project explores real-world datasets using box plot visualization and correlation analysis techniques.

The analysis is performed on:

* Video Game Reviews Dataset
* Health Monitoring Dataset

## Objectives

* Visualize data distributions using box plots (median, IQR, outliers)
* Apply appropriate correlation techniques (Pearson, Spearman, Point-Biserial)
* Interpret statistical relationships and patterns

## Datasets

1. Video Game Reviews

   * Hours Played
   * Review Score

2. Health Monitoring

   * Age
   * Blood Pressure
   * Smoking Status

## Methods Used

* Box Plot Visualization
* Correlation Analysis:

  * Pearson Correlation (continuous variables)
  * Point-Biserial Correlation (binary vs continuous)

## Key Insights

* Relationship between gameplay hours and review scores
* Impact of smoking on blood pressure
* Variability and outliers across age groups
